# Matias Speroni Portfolio

![CI](https://github.com/ElSenpaiSAMA/NextJS-Portfolio/actions/workflows/ci.yml/badge.svg)


Personal portfolio developed with Next.js 16, React 19 and TypeScript.

Technologies Used:

-  Next.js, React, TypeScript, Tailwind CSS

API:
- Formspree

Architecture:

The project uses Next.js App Router with the following structure:

- `app/`: App Router pages and layouts
- `app/components/`: Reusable components
- `app/data/`: Static data (projects)
- `app/styles/`: Separate style sheets
- `public/`: Static files

 Installation:

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`

Deployment:

The project is optimized for Vercel. To deploy:

1. Connect your repository to Vercel
2. Deploy automatically

Development:

- `npm run dev`: Start development server
- `npm run build`: Build for production
- `npm run lint`: Run ESLint
